Chapter 1: Linux Command Line Essentials

Command line proficiency accelerates penetration testing workflows

The Linux command line is your primary interface for penetration testing activities. Mastering these fundamental commands will significantly improve your efficiency and effectiveness.

File System Navigation

Basic Navigation Commands

pwd - Print working directory

ls -la - List files with detailed information

cd /path/to/directory - Change directory

find / -name "*.conf" 2>/dev/null - Find configuration files

locate filename - Quick file location

File Manipulation

tail -f /var/log/auth.log - Monitor log files

head -n 20 file.txt - Display first 20 lines

wc -l file.txt - Count lines in file

Chapter 2: File Permissions and Security

Understanding file permissions is crucial for privilege escalation

Linux file permissions control access to files and directories. Understanding permission models is essential for identifying privilege escalation opportunities during penetration tests.

Permission Types

r Read - View file contents or list directory
w Write - Modify file or directory contents
x Execute - Run file or access directory

Special Permissions

s SUID/SGID - Run with owner/group privileges
t Sticky bit - Restrict deletion in directories

Chapter 5: Text Processing and Analysis

Text processing skills are vital for analyzing logs and scan results

Text processing capabilities allow you to efficiently analyze large datasets, log files, and tool outputs during penetration testing activities.

Advanced Grep Usage

Chapter 6: Shell Scripting for Automation

Shell scripting enables automation of repetitive penetration testing tasks

Shell scripting allows you to automate repetitive tasks, create custom tools, and streamline your penetration testing workflow.

Basic Script Structure

#!/bin/bash

# Port Scanner Script
target=$1
if [ -z "$target" ]; then
    echo "Usage: $0 "
    exit 1
fi

echo "Scanning $target..."
for port in {1..1000}; do
    timeout 1 bash -c "echo >/dev/tcp/$target/$port" 2>/dev/null && 
    echo "Port $port is open"
done
