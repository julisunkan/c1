Linux Fundamentals for Penetration Testing

Master essential Linux commands, file systems, and tools required for penetration testing

Introduction

Linux is the foundation of most penetration testing environments. This module covers essential Linux concepts, commands, and tools that every penetration tester must master. From basic navigation to advanced system administration, you'll build the Linux skills necessary for effective security testing.

Learning Objectives

  • Navigate the Linux file system efficiently
  • Master essential command-line tools
  • Understand Linux permissions and security model
  • Learn process management and system monitoring
  • Configure network settings and services
  • Use text processing and scripting tools
  • Implement security hardening techniques

Essential Linux Commands

File System Navigation

# Basic navigation
pwd                 # Print working directory
ls -la             # List files with details
cd /path/to/dir    # Change directory
cd ~               # Go to home directory
cd ..              # Go up one directory
find / -name "file" # Search for files

File Operations

# File manipulation cp source dest # Copy files mv source dest # Move/rename files rm -rf directory # Remove files/directories chmod 755 file # Change permissions chown user:group file # Change ownership ln -s target link # Create symbolic link

Text Processing

# Text manipulation cat file.txt # Display file content grep "pattern" file # Search in files sed 's/old/new/g' file # Replace text awk '{print $1}' file # Process columns sort file.txt # Sort lines uniq file.txt # Remove duplicates

Linux File System Structure

/bin & /usr/bin

Essential command binaries and user programs

/etc

System configuration files

/var

Variable data files, logs, databases

/home

User home directories

/tmp

Temporary files

/proc

Process and kernel information

Process Management

# Process control ps aux # List all processes top # Real-time process viewer htop # Enhanced process viewer kill PID # Terminate process killall process # Kill by name nohup command & # Run in background jobs # List background jobs

Network Commands

# Network tools ifconfig # Network interface configuration ip addr show # Show IP addresses netstat -tulpn # Show network connections ss -tulpn # Modern netstat replacement ping host # Test connectivity traceroute host # Trace network path nslookup domain # DNS lookup

System Information

# System details uname -a # System information whoami # Current user id # User and group IDs uptime # System uptime df -h # Disk usage free -h # Memory usage lscpu # CPU information

File Permissions

Understanding Linux permissions is crucial for security testing:

# Permission format: rwxrwxrwx (owner, group, others) # r=4, w=2, x=1 chmod 755 file # rwxr-xr-x chmod 644 file # rw-r--r-- chmod +x script # Add execute permission chown user:group file # Change ownership

Log Files and Monitoring

# Important log locations /var/log/syslog # System messages /var/log/auth.log # Authentication logs /var/log/apache2/ # Web server logs /var/log/nginx/ # Nginx logs # Log monitoring tail -f /var/log/syslog # Follow log in real-time grep "error" /var/log/* # Search for errors journalctl -f # Systemd journal

Package Management

Debian/Ubuntu (APT)

apt update apt install package apt remove package apt search keyword

Red Hat/CentOS (YUM/DNF)

yum update yum install package yum remove package yum search keyword

Text Editors

Vi/Vim Basics

# Vim commands vim filename # Open file i # Insert mode :w # Save file :q # Quit :wq # Save and quit :q! # Quit without saving /pattern # Search dd # Delete line

Shell Scripting Basics

#!/bin/bash # Basic script structure echo "Hello, World!" read -p "Enter your name: " name echo "Hello, $name" # Variables var="value" echo $var # Conditions if [ "$var" == "value" ]; then echo "Match found" fi # Loops for i in {1..5}; do echo $i done

Environment Variables

# Common environment variables echo $PATH # Command search path echo $HOME # User home directory echo $USER # Current username export VAR=value # Set environment variable env # List all environment variables

Archive and Compression

# tar commands tar -cvf archive.tar files/ # Create archive tar -xvf archive.tar # Extract archive tar -czvf archive.tar.gz files/ # Create compressed archive tar -xzvf archive.tar.gz # Extract compressed archive # Other compression gzip file.txt # Compress file gunzip file.txt.gz # Decompress file zip archive.zip files/ # Create ZIP archive unzip archive.zip # Extract ZIP archive

Security-Related Commands

# Security tools sudo command # Execute as root su username # Switch user passwd # Change password last # Show last logins w # Show logged-in users history # Command history
Security Note: Always use sudo instead of logging in as root directly. This provides better audit trails and reduces the risk of accidental system damage.
