Module 11: Advanced Attack Techniques

Container Security Testing

Container technologies like Docker and Kubernetes present unique attack surfaces and security challenges.

Docker Attack Vectors

  • • Container Escape: Breaking out of container isolation
  • • Privileged Containers: Elevated permission abuse
  • • Insecure Images: Vulnerable base images and secrets
  • • Docker Daemon: API exposure and socket access
  • • Volume Mounts: Host filesystem access
  • • Network Bridges: Container-to-host communication

Kubernetes Attacks

  • • API Server Attacks: Unauthorized cluster access
  • • RBAC Bypass: Permission escalation
  • • Pod Security: Insecure pod configurations
  • • Service Account Abuse: Token theft and reuse
  • • Secrets Management: Credential exposure
  • • Admission Controllers: Policy bypass techniques

Active Directory Domain Dominance

Advanced Active Directory attacks focus on achieving domain administrator privileges and maintaining persistence.

Kerberos Attacks

  • • Golden Ticket: Forge TGT with krbtgt hash
  • • Silver Ticket: Forge service tickets
  • • Diamond Ticket: Modify existing TGTs
  • • Sapphire Ticket: Cross-realm ticket attacks
  • • Skeleton Key: Backdoor authentication
  • • DCShadow: Rogue domain controller

Certificate Services Attacks

  • • ESC1-ESC8: Certificate template abuse
  • • NTLM Relay: Certificate-based authentication
  • • PetitPotam: Force NTLM authentication
  • • PrinterBug: Coerce authentication
  • • ShadowCredentials: Key trust abuse
  • • UnPAC the Hash: S4U2self delegation
