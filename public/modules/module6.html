Chapter 4: Search Engine Intelligence

Advanced search techniques reveal hidden information and sensitive data

Search engine dorking uses advanced search operators to discover sensitive information, configuration files, and security vulnerabilities exposed through web indexing.

Google Dorking Techniques

Basic Operators

site:target.com - Search within specific domain

filetype:pdf site:target.com - Find PDF documents

intitle:"index of" site:target.com - Directory listings

inurl:admin site:target.com - Admin interfaces

Sensitive Information

filetype:sql "INSERT INTO" site:target.com - Database dumps

filetype:log site:target.com - Log files

intitle:"config" filetype:txt - Configuration files

"password" filetype:txt site:target.com - Password files
